#  Satellite Orbit Simulator

This project is a Python-based satellite orbit simulator that models and visualizes orbital motion using fundamental physics and computational techniques.

##  Features
- Simulates satellite motion around Earth
- Visualizes orbital paths using graphs/animation
- Demonstrates real-world orbital mechanics concepts

##  Concepts Used
- Newton’s Laws of Motion
- Gravitational Forces
- Orbital Mechanics
- Numerical Simulation

##  Tech Stack
- Python
- NumPy
- Matplotlib

##  Future Improvements
- Real-time satellite tracking using APIs
- 3D visualization
- Collision prediction system
